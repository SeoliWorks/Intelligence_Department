# Intelligence_Department

外務省における「インテリジェンス（諜報・情報収集・分析）」は、日本の国家安全保障や外交政策の意思決定において極めて重要な役割を果たしています。映画のような「スパイ活動」とは異なり、外交官としての身分を持ちながら、合法的な手段や人脈を駆使して行う活動が主体です。

以下に、組織構造、収集手法、役割、そして課題について体系的にまとめました。

---

### 1. 組織構造：情報の「司令塔」

外務省のインテリジェンス機能の中核を担っているのは、本省にある以下の組織です。

**■ 国際情報統括官組織（Intelligence and Analysis Service: IAS）**
かつての「情報調査局」が改組され、1993年に設置されました。政策立案部門から独立させることで、**「政策による情報の歪曲（Policy contamination）」**を防ぎ、客観的な分析を行うことを目的としています。

トップは**国際情報統括官**（局長級）であり、その下に以下の4つの課室が配置されています。

* **国際情報統括官付 第1国際情報官室:** 安全保障、テロ、非拡散、科学技術など、特定のテーマ（機能別）を担当。
* **国際情報統括官付 第2国際情報官室:** アジア大洋州、北米、中南米など（地域別）を担当。
* **国際情報統括官付 第3国際情報官室:** 欧州、中東、アフリカなど（地域別）を担当。
* **国際情報統括官付 第4国際情報官室:** データ分析、公開情報（OSINT）の収集・分析、情報保全を担当。

### 2. 情報収集の手法（Collection）

外務省の最大のアセット（資産）は、世界中に張り巡らされた在外公館（大使館・総領事館）のネットワークです。

* **HUMINT（ヒューミント：人的情報）の重視**
* 外務省インテリジェンスの真骨頂です。外交官が各国の政府高官、政治家、ジャーナリスト、シンクタンク研究者などと接触し、会話の中から「表に出ない意図」や「内部情報」を入手します。
* **「メモ魔」の文化:** 外交官は接触内容を詳細な「公電（電報）」として本省に報告します。これが一次情報となります。


* **OSINT（オシント：公開情報）**
* 現地の新聞、テレビ、インターネット、政府発表資料などの公開情報を収集・分析します。言語の専門家（地域専門職）が、現地の微妙なニュアンスを読み解きます。


* **IMINT（イミント：画像情報）との連携**
* 外務省自体は偵察衛星を持ちませんが、内閣衛星情報センターなどから提供される衛星画像の分析を政策判断に活用します。



### 3. 分析と配布（Analysis & Dissemination）

収集された情報は、そのままでは単なる「Information」ですが、分析官が背景知識と組み合わせて分析・評価することで「Intelligence」に昇華されます。

* **分析のプロセス:**
在外公館から上がってきた「公電」やOSINT情報を、国際情報統括官組織の分析官が精査します。
* **配布先:**
分析結果は、外務大臣、首相官邸（内閣総理大臣、官房長官）、国家安全保障会議（NSC）などに報告され、外交交渉や安全保障政策の基礎資料となります。

### 4. 日本のインテリジェンス・コミュニティ（IC）における位置づけ

日本の情報機関は「内閣情報調査室（内調）」をハブとして連携していますが、外務省はその中でも**「対外情報の主役」**です。

| 組織 | 主な役割・特徴 |
| --- | --- |
| **外務省（国際情報統括官）** | **対外・外交情報（HUMINT中心）。** 国際情勢の分析。 |
| **内閣情報調査室（CIRO）** | 官邸直属。各省庁からの情報を集約・整理し、総理に報告。国内・メディア情報にも強い。 |
| **防衛省（情報本部）** | 軍事情報。電波傍受（SIGINT）や衛星画像など技術的な収集に強み。 |
| **警察庁（警備局）** | 治安情報。テロ対策、カウンターインテリジェンス（防諜）。 |

### 5. 特筆すべきアセットと人材

* **専門調査員:**
在外公館には、外交官だけでなく、特定の地域や分野に精通した外部の研究者などが「専門調査員」として派遣され、深い分析を行います。
* **防衛駐在官:**
自衛官が外務事務官（外交官）の身分を併せ持ち、大使館に駐在します。現地の軍事情報の収集や、各国の軍人とのリエゾン（連絡）を行います。

### 6. 課題と論点

外務省のインテリジェンスには、構造的な制約や課題も指摘されています。

* **「スパイ組織」ではない:**
CIAやMI6のような非合法活動（盗聴、買収、潜入など）を行う権限や組織力はありません。あくまで外交特権の範囲内での活動が基本です。
* **防諜（カウンターインテリジェンス）の弱さ:**
情報を取ることは得意でも、情報漏洩を防ぐ、あるいは他国の工作を防ぐ権限（法執行権）は警察や公安調査庁にあり、外務省単独では限界があります。
* **セキュリティ・クリアランス:**
経済安全保障推進法の制定などにより進展していますが、同盟国（ファイブ・アイズ等）との機微な情報共有において、日本のセキュリティ体制（適性評価制度）の強化が常に求められています。

### まとめ

外務省におけるインテリジェンスとは、**「世界中の大使館網と外交官の人間力を駆使した、政策決定のための高度な状況分析」**と言えます。軍事的な諜報活動とは異なり、外交交渉を有利に進め、日本の国益を守るための「知的な武器」としての側面が強いのが特徴です。

### 2. 各国の詳細な特徴

#### 🇺🇸 米国：国務省 情報調査局 (INR)

*Bureau of Intelligence and Research*

**「巨大な情報コミュニティにおける『良心』」**
米国にはCIAやNSAなど巨大な情報機関がありますが、国務省INRはそれらとは一線を画す**「分析専門」**の小規模組織です。

* **役割:** スパイ活動（収集）は一切行わず、CIAやNSAが集めた情報、外交公電、公開情報を総合的に分析し、国務大臣（国務長官）に報告します。
* **特徴:** **「独立性」**を極めて重視します。CIAが時の政権の意向を気にして分析を歪めるリスク（イラク戦争時の大量破壊兵器問題など）がある中で、INRは**「嫌な事実でも直言する（Speaking truth to power）」**文化で知られます。ベトナム戦争やイラク戦争の際、CIAの主流派見解に対して異論（Dissent）を唱え、後に正しかったことが証明された事例が多くあります。
* **人材:** 外交官のローテーションではなく、特定の地域を何十年も研究し続ける**「シビル・サービス（公務員研究職）」**が主力を担います。

#### 🇬🇧 英国：外務・英連邦・開発省 (FCDO)

*Foreign, Commonwealth and Development Office*

**「スパイ組織の『オーナー』としての外務省」**
英国の最大の特徴は、秘密情報部（**MI6**）と政府通信本部（**GCHQ**）という強力な実働部隊が、**外務大臣の直接の指揮下**にあることです。

* **リサーチ・アナリスト (Research Analysts):**
FCDO内部には、英国のトップレベルの地域研究者が集まる「リサーチ・アナリスト」部門があります。彼らは大学教授並みの専門知識を持ち、長期的な視点で情勢を分析します。
* **JIC（合同情報委員会）との関係:**
英国では、各省庁の情報が内閣府にある**JIC**に集約され、「統一見解」が作られます。FCDOのアナリストやMI6の情報は、このJICプロセスの最も重要な構成要素となります。
* **強み:** 政策立案（外務省）と情報収集（MI6）のトップが同じ大臣の下にいるため、外交目標に合わせて柔軟にスパイ活動の優先順位を決めることができます。

#### 🇯🇵 日本：外務省 国際情報統括官組織

**「外交官＝インテリジェンス・オフィサー」**
日本は、米国のような巨大な独立分析機関も、英国のような外相直轄のスパイ組織も持ちません。その代わり、**外交官自身が収集と分析の両方を行う**のが特徴です。

* **ハイブリッド型:**
通常の外交官が、赴任先で人脈を作り情報収集（HUMINT）を行います。そして本省に戻れば、今度は分析官として政策立案を補佐します。現場感覚を持った分析ができる反面、専門家の育成（情報のプロ化）が長年の課題とされてきました。
* **構造的な制約:**
CIAやMI6のような対外情報機関（法執行権や工作権限を持つ組織）を持たないため、どうしても「相手の善意や、公開情報、公式な会談」に依存する割合が高くなります。これを補うために「専門調査員」などの外部専門家を活用しています。

### 3. 比較から見える日本の課題と方向性

この比較から、日本の外務省インテリジェンスが直面している論点が見えてきます。

1. **「分析の独立性」の確保 (対 米国INR)**
* 日本の外交官は政策立案も行うため、「自らが進めたい政策に都合の良い情報ばかり集めてしまう」リスク（Policy contamination）が常にあります。INRのように、**「分析に特化した専門職（ノンキャリアの分析官など）」**の地位をどう確立するかが課題です。


2. **「ヒューミント」の質的転換 (対 英国MI6)**
* 英国のように強力な諜報機関を外務省の下に持たない日本は、外交官個人の「人間力」に頼る部分が大きいです。近年は、防衛駐在官（自衛隊出身者）との連携を深めたり、民間の商社や企業との情報共有を強化したりすることで、**「オールジャパン」での情報収集力**を高めようとしています。



### まとめ

* **米国**は、あえて組織を分け、相互に批判させることで質を高める**「競争モデル」**。
* **英国**は、強力なスパイ組織を外務省の一部として統合運用する**「統合モデル」**。
* **日本**は、外交官が一人二役をこなす**「現場一体モデル」**。

それぞれに良し悪しがありますが、日本はいかにして「現場一体」の強みを活かしつつ、「分析の客観性」を担保するかが、インテリジェンス能力向上の鍵となります。
